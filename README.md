## Install

Please refer to [MMDetection v1.0rc1](https://github.com/open-mmlab/mmdetection/tree/v1.0rc1) for installation.

**Recommended environment**

```
Python 3.6+
Pytorch 1.1.0
torchvision 0.2.1
pytorch_transformers 1.1.0
mmcv 0.2.13
Polygon3
opencv-python 4.4.0
```
